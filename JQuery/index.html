La synthese General d'une instruction JQuery est:

  • $(sel).action ou Sel est un element HTML ou un selecteur CSS;
  • action est la fonction JQuery a appliquer sur sel.
Selecteur CSS JQuery Description
Element de balise exemple de Balise p $('p') Selectionner tous les paragraphes du document
identifiant nomId $('#nomId') Selectionner l'element Unique du document dont l'attrivut id est egal a nomId
classe maClasse $('.maClasse') Selectionner tous les elements du document dont l'attribut est egal a maClasse
JS JQuery
value val()
innerHTML html()
innerText - textContent text()
value val()
exemple
$('li : first') recupere le 1er element d'une list non ordonnee
$('li : last') recupere le dernier element d'une list non ordonnee
$('li : eq(2)') recupere l'element dans la 3e position d'une list non ordonnee
$('li : gt(n)') recupere l'element dont la position est plus grand a n
$('li : even') recupere tous les elements d'indice pairs
$('li : odd') recupere tous les elements d'indice impairs
$('li : not(.maclasse)') tous les elements saud ceux de l'attribut class = maclasse
